## Installation:

- Create new dir and change your current dir to `project`:
```
mkdir proejct
cd project 
```
- Clone repository to `project` dir: 
```bash
git clone git@github.com:djeck1432/<project_name>.git
```
- Run docker-compose:
```
docker-compose up -d
```
- Stop docker-compose:
```bash
docker-compose down
```
